# FastAPI Backend

## 📌 프로젝트 개요
이 프로젝트는 FastAPI 기반의 백엔드 서비스입니다.  
RESTful API를 제공하며, 인증/인가, 데이터베이스 연동, 배포 환경을 고려하여 설계되었습니다.

---
